PAY FOR PARKING

Annotations: The overall goal of this project was to transition the original parking violation payment page into a more user friendly, interactive, and visually improved web design. In order to do this, the primary goal was to implement two things: buttons and a navigation bar. The buttons appear larger for the user to see and interpret. Also, they have a hover feature in which it changes the color of the button as the user moves over it. The navigation bar serves as a pagination or bootstrap. Although there was a navigation bar in the original page, this one visuyally improves as the user hovers it as well.
